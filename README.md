# 🚀 HandPPT - 隔空手势 PPT 翻页助手

这是一个基于 Python 和 AI 技术开发的小工具。无需任何硬件，只要有摄像头，挥挥手就能控制你的 PPT 翻页。

### 🌟 功能特点
- **去 AI 味**：注释全是中文，逻辑清晰，适合新手学习。
- **防止误触**：特别设计了屏幕触发区和 1.5 秒的冷却时间，体验更流畅。
- **简单好用**：手掌张开，在画面上方左挥上一页，右挥下一页。

### 🛠️ 快速开始
1.  **下载模型**：从 Google 官方下载 `hand_landmarker.task` 模型文件，并将其放在本项目的根目录下（和 `main.py` 在一起）。
2.  **安装依赖**：
    ```bash
    pip install -r requirements.txt
    ```
3.  **运行程序**：
    ```bash
    python main.py
    ```
